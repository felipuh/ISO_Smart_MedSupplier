# Compliance Notes

ISO Smart MedSupplier se diseña con controles alineados a buenas prácticas para entornos regulados, incluyendo trazabilidad, control de acceso, segregación shared/private, versionado documental y audit trail.

El producto puede venderse standalone o integrado con Iso Smart. En ambos casos AdminApps mantiene la autoridad sobre organizaciones cliente, usuarios, roles, suscripciones y habilitación del módulo `MEDSUPPLIER`. En modo standalone comercial debe conservar controles de acceso, audit trail, trazabilidad documental, exportabilidad y validación de registros sin depender de que el cliente haya comprado Iso Smart.

## Lenguaje permitido

- compliance-ready
- audit-ready
- validation-ready
- Part 11-ready controls
- QMS-ready

## Lenguaje no permitido

- Cumple automáticamente FDA, MDR o ISO 13485.
- Sistema certificado o validado sin evidencia formal.

## Normativas consideradas

- 21 CFR Part 11
- ISO 13485
- ISO 9001
- ISO 14971
- MDR / Regulation EU 2017/745
- OWASP

El cumplimiento formal requiere procesos, SOPs, capacitación, validación, controles operativos y revisión de uso real.
