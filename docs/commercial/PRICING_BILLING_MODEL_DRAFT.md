# Pricing And Billing Model Draft

Date: 2026-06-29

This is a commercial draft, not a final price book.

## Packaging Options

| Package | Description |
| --- | --- |
| Controlled Demo | Short evaluation with seeded or customer-approved non-sensitive data. |
| Paid Pilot | Limited-scope project with real workflows and defined success criteria. |
| Controlled Enterprise MVP | Deeper configuration, integrations, and validation-ready documentation. |
| Production-Like Non-Regulated | Controlled production-like deployment for non-regulated use cases. |
| Validation Support Package | Documentation and evidence package support for customer/company validation. |

## Billing Dimensions

- Organization count.
- Supplier/customer account count.
- Authorized user count.
- Product bundle: MedSupplier standalone or MedSupplier + ISO Smart.
- Support tier.
- Deployment option.
- Validation/documentation support scope.

## Suggested Model

- Setup fee for onboarding, configuration, training, and environment preparation.
- Monthly or annual subscription for product access.
- Optional professional services for validation package support, integrations, data migration, and SOP alignment.

## Billing Authority

AdminApps remains the system of record for product entitlements, billing state, suspension, and product access decisions.
