# Support Model Draft

Date: 2026-06-29

## Channels

- Email or ticketing queue for pilots and enterprise MVPs.
- Scheduled review meetings for pilot success criteria.
- Emergency contact path only for contracted production-like deployments.

## Responsibilities

Smart3AI / Product Team:

- Application triage.
- Defect analysis.
- Release notes.
- Deployment guidance.
- Validation package support if contracted.

Customer:

- User access approval.
- SOP ownership.
- Validation protocol approval.
- Infrastructure approvals.
- Data quality and data classification decisions.

## Cadence

- Demo: ad hoc.
- Paid pilot: weekly review.
- Enterprise MVP: weekly or biweekly review.
- Production-like non-regulated: monthly service review plus incident review as needed.
