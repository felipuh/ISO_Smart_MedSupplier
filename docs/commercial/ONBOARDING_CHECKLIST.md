# Onboarding Checklist

Date: 2026-06-29

## Commercial

- [ ] Confirm package: demo, pilot, enterprise MVP, or production-like non-regulated.
- [ ] Confirm scope and success criteria.
- [ ] Confirm support tier.
- [ ] Confirm pricing/billing owner.

## Technical

- [ ] Confirm tenant organization.
- [ ] Configure AdminApps product entitlement.
- [ ] Configure users and roles.
- [ ] Configure MedSupplier customer accounts.
- [ ] Confirm allowed origins and hosts.
- [ ] Confirm backup/restore plan.
- [ ] Confirm rollback plan.

## Quality / Validation

- [ ] Review validation package.
- [ ] Review role permission matrix.
- [ ] Review data classification.
- [ ] Identify SOP impact.
- [ ] Confirm whether formal validation is in scope.
- [ ] Complete human approval template.

## Demo / Pilot Data

- [ ] Use seeded demo data or approved non-sensitive data.
- [ ] Do not upload regulated production data unless scope and controls are approved.
