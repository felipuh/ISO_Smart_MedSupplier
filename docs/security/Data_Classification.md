# Data Classification

- `public_shared`: controlled shared information suitable for broad account visibility.
- `customer_shared`: Supplier-Customer operational information for scoped Customer accounts.
- `supplier_private`: internal Supplier information not visible to Customer.
- `supplier_confidential`: highly restricted Supplier information.
- `regulated_evidence`: evidence intended for audit-ready packages.
- `audit_only`: audit trail or review-only information.
- `archived`: retained non-current information.
- `obsolete`: superseded or obsolete controlled record.
